# Toggles

> Toggle buttons and toggle groups for binary and multi-option selection

**Browse**: https://21st.dev/s/toggle

## Components

Browse the full collection at https://21st.dev/s/toggle

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- View mode toggles (grid/list)
- Feature on/off toggles
- Text formatting toggles
