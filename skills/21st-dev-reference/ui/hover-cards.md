# Hover Cards

> Cards that appear on hover showing preview content or user profiles

**Browse**: https://21st.dev/s/hover-card

## Top Components

| Component | Author | Description | Deps | Install |
|-----------|--------|-------------|------|---------|
| direction-aware-hover | aceternity | direction aware hover component | framer-motion | `npx shadcn@latest add "https://21st.dev/r/aceternity/direction-aware-hover"` |
| link-preview | aceternity | link preview component | qss, framer-motion, @radix-... | `npx shadcn@latest add "https://21st.dev/r/aceternity/link-preview"` |

## When to Use

- User profile preview on hover
- Link preview tooltips
- Content snippet previews
