# Skeletons

> Loading skeleton placeholders that mimic content layout while data loads

**Browse**: https://21st.dev/s/skeleton

## Components

Browse the full collection at https://21st.dev/s/skeleton

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- Content loading placeholders
- Card/list skeleton screens
- Dashboard loading states
