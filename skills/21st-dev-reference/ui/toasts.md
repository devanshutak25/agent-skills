# Toasts

> Toast notification components with actions, auto-dismiss, and stacking

**Browse**: https://21st.dev/s/toast

## Components

Browse the full collection at https://21st.dev/s/toast

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- Action success notifications
- Error alert toasts
- Undo action notifications
