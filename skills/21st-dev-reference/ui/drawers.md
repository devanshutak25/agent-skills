# Drawers

> Slide-out drawer panels from edges of the screen

**Browse**: https://21st.dev/s/drawer

## Components

Browse the full collection at https://21st.dev/s/drawer

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- Mobile navigation panels
- Cart/preview side panels
- Detail drawers on desktop
