# Switches

> Toggle switch components with animations and custom themes

**Browse**: https://21st.dev/s/switch

## Components

Browse the full collection at https://21st.dev/s/switch

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- Feature toggles in settings
- Dark mode switches
- Notification preference toggles
