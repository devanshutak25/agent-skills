## Components

Browse the full collection at https://21st.dev/s/combobox

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- Searchable dropdown selectors
- Tag input with suggestions
- Country/city selection fields
