# Footers

> Page footers with navigation links, social media, newsletters, and branding

**Browse**: https://21st.dev/s/footer

## Components

Browse the full collection at https://21st.dev/s/footer

Install any component with:
```
npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
```

## When to Use

- Site-wide footer with link columns
- Footer with newsletter and social links
- Minimal copyright footers
