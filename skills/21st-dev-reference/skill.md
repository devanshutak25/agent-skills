# 21st.dev Component Reference

> Community registry of thousands of React UI components — shadcn/ui-compatible, Tailwind CSS + Radix UI

## Platform Overview

- **What**: Open marketplace for React UI components (anyone can publish)
- **Stack**: React, TypeScript, Tailwind CSS, Radix UI — fully shadcn/ui-compatible
- **Key Publishers**: Aceternity UI, Magic UI, Origin UI, Prism UI, serafimcloud
- **Install any component**:
  ```
  npx shadcn@latest add "https://21st.dev/r/{author}/{component-slug}"
  ```

## Integration Notes

- All components are **shadcn/ui-compatible** — they install into your `components/ui/` directory
- Requires: React 18+, Tailwind CSS, TypeScript (recommended)
- Common dependencies: `framer-motion`, `lucide-react`, `@radix-ui/*`
- After install: import from `@/components/ui/{component-name}`
- Some components need additional setup (e.g., Tailwind config for animations)
- The `npx shadcn@latest add` command handles dependency installation automatically
