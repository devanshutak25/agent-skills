# Folder

> Interactive folder opens to reveal nested content smooth motion.

**Category**: Components  
**Docs**: https://reactbits.dev/components/folder

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number` | `2` | Element size |
| `color` | `string` | `"#5227FF"` | Primary color |
| `className` | `string` | `"custom-folder"` | Additional CSS classes |

## Usage

```jsx
import Folder from './Folder'

<div style={{ height: '600px', position: 'relative' }}>
  <Folder size={2} color="#5227FF" className="custom-folder" />
</div>
```
