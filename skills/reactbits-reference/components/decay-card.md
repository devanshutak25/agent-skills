# DecayCard

> Hover parallax effect that disintegrates the content of a card.

**Category**: Components  
**Docs**: https://reactbits.dev/components/decay-card

## Dependencies

```bash
npm install gsap
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `number` | `200` | Width |
| `height` | `number` | `300` | Height |
| `image` | `string` | `"https://picsum.photos/300/400?grayscale"` | Image |

## Usage

```jsx
import DecayCard from './DecayCard';

<DecayCard width={200} height={300} image="https://picsum.photos/300/400?grayscale">
  <h2>Decay<br/>Card</h2>
</DecayCard>
```
