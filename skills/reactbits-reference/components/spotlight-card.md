# SpotlightCard

> Dynamic spotlight follows cursor casting gradient illumination.

**Category**: Components  
**Docs**: https://reactbits.dev/components/spotlight-card

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `"custom-spotlight-card"` | Additional CSS classes |
| `spotlightColor` | `string` | `"rgba(0, 229, 255, 0.2)"` | Spotlight Color |

## Usage

```jsx
import SpotlightCard from './SpotlightCard';
  
<SpotlightCard className="custom-spotlight-card" spotlightColor="rgba(0, 229, 255, 0.2)">
  // Content goes here
</SpotlightCard>
```

## Suggested Use Cases

- Modern web apps
