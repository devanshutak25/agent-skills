# GlitchText

> RGB split and distortion glitch effect with jitter effects.

**Category**: Text Animations  
**Docs**: https://reactbits.dev/text-animations/glitch-text

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `speed` | `number` | `1` | Animation speed |
| `enableShadows` | `boolean` | `true` | Enable Shadows |
| `enableOnHover` | `boolean` | `true` | Enable On Hover |
| `className` | `string` | `"custom-class"` | Additional CSS classes |

## Usage

```jsx
import GlitchText from './GlitchText';
  
<GlitchText
  speed={1}
  enableShadows={true}
  enableOnHover={true}
  className='custom-class'
>
  React Bits
</GlitchText>
```

## Suggested Use Cases

- Hero section headlines
- Landing pages
