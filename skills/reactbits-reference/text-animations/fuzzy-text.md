# FuzzyText

> Vibrating fuzzy text with controllable hover intensity.

**Category**: Text Animations  
**Docs**: https://reactbits.dev/text-animations/fuzzy-text

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `baseIntensity` | `number` | `0.2` | Base Intensity |
| `hoverIntensity` | `any` | `hoverIntensity` | Hover Intensity |
| `enableHover` | `any` | `enableHover` | Enable Hover |

## Usage

```jsx
import FuzzyText from './FuzzyText';
  
<FuzzyText 
  baseIntensity={0.2} 
  hoverIntensity={hoverIntensity} 
  enableHover={enableHover}
>
  404
</FuzzyText>
```

## Suggested Use Cases

- Hero section headlines
- Interactive text
- Landing pages
