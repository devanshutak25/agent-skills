# StarBorder

> Animated star / sparkle border orbiting content with twinkle pulses.

**Category**: Animations  
**Docs**: https://reactbits.dev/animations/star-border

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `string` | `"button"` | As |
| `className` | `string` | `"custom-class"` | Additional CSS classes |
| `color` | `string` | `"cyan"` | Primary color |
| `speed` | `string` | `"5s"` | Animation speed |

## Usage

```jsx
import StarBorder from './StarBorder'
  
<StarBorder
  as="button"
  className="custom-class"
  color="cyan"
  speed="5s"
>
  // content
</StarBorder>
```

## Suggested Use Cases

- Scroll-triggered reveals
- Decorative accents
- Creative landing pages
