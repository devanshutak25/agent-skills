# ClickSpark

> Creates particle spark bursts at click position.

**Category**: Animations  
**Docs**: https://reactbits.dev/animations/click-spark

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `sparkColor` | `string` | `"#fff"` | Spark color |
| `sparkSize` | `number` | `10` | Spark size |
| `sparkRadius` | `number` | `15` | Spark Radius |
| `sparkCount` | `number` | `8` | Number of sparks |
| `duration` | `number` | `400` | Animation duration |

## Usage

```jsx
import ClickSpark from './ClickSpark';

<ClickSpark
  sparkColor='#fff'
  sparkSize={10}
  sparkRadius={15}
  sparkCount={8}
  duration={400}
>
  {/* Your content here */}
</ClickSpark>
```
