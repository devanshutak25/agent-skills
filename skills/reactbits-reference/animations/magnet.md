# Magnet

> Elements magnetically ease toward the cursor then settle back with spring physics.

**Category**: Animations  
**Docs**: https://reactbits.dev/animations/magnet

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `padding` | `number` | `50` | Padding |
| `disabled` | `boolean` | `false` | Disabled |
| `magnetStrength` | `number` | `50` | Magnet Strength |

## Usage

```jsx
import Magnet from './Magnet'

<Magnet padding={50} disabled={false} magnetStrength={50}>
  <p>Star React Bits on GitHub!</p>
</Magnet>
```

## Suggested Use Cases

- Custom cursor effects
- Creative landing pages
