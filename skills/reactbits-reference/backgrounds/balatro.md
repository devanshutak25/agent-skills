# Balatro

> The balatro shader, fully customizalbe and interactive.

**Category**: Backgrounds  
**Docs**: https://reactbits.dev/backgrounds/balatro

## Dependencies

```bash
npm install ogl
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isRotate` | `boolean` | `false` | Is Rotate |
| `mouseInteraction` | `boolean` | `true` | Mouse Interaction |
| `pixelFilter` | `number` | `700` | Pixel Filter |

## Usage

```jsx
import Balatro from './Balatro';
  
<Balatro
  isRotate={false}
  mouseInteraction={true}
  pixelFilter={700}
/>
```

## Suggested Use Cases

- Hero backgrounds
- Landing page sections
