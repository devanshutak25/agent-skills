# Iridescence

> Slick iridescent shader with shifting waves.

**Category**: Backgrounds  
**Docs**: https://reactbits.dev/backgrounds/iridescence

## Dependencies

```bash
npm install ogl
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `array` | `[1, 1, 1]` | Primary color |
| `mouseReact` | `boolean` | `false` | Mouse React |
| `amplitude` | `number` | `0.1` | Wave amplitude |
| `speed` | `number` | `1.0` | Animation speed |

## Usage

```jsx
import Iridescence from './Iridescence';
  
<Iridescence
  color={[1, 1, 1]}
  mouseReact={false}
  amplitude={0.1}
  speed={1.0}
/>
```

## Suggested Use Cases

- Hero backgrounds
- Elegant backgrounds
- Landing page sections
