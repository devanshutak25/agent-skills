# Silk

> Smooth waves background with soft lighting.

**Category**: Backgrounds  
**Docs**: https://reactbits.dev/backgrounds/silk

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `speed` | `number` | `5` | Animation speed |
| `scale` | `number` | `1` | Scale multiplier |
| `color` | `string` | `"#7B7481"` | Primary color |
| `noiseIntensity` | `number` | `1.5` | Noise Intensity |
| `rotation` | `number` | `0` | Rotation |

## Usage

```jsx
import Silk from './Silk';

<Silk
  speed={5}
  scale={1}
  color="#7B7481"
  noiseIntensity={1.5}
  rotation={0}
/>
```

## Suggested Use Cases

- Hero backgrounds
- Elegant backgrounds
- Landing page sections
