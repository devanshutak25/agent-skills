# LetterGlitch

> Matrix style letter animation.

**Category**: Backgrounds  
**Docs**: https://reactbits.dev/backgrounds/letter-glitch

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `glitchSpeed` | `number` | `50` | Glitch Speed |
| `centerVignette` | `boolean` | `true` | Center Vignette |
| `outerVignette` | `boolean` | `false` | Outer Vignette |
| `smooth` | `boolean` | `true` | Smooth |

## Usage

```jsx
import LetterGlitch from './LetterGlitch';
  
<LetterGlitch
  glitchSpeed={50}
  centerVignette={true}
  outerVignette={false}
  smooth={true}
/>
```

## Suggested Use Cases

- Hero backgrounds
- Landing page sections
